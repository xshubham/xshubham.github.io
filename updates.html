students_database.sql 👩‍🎓👨‍🎓

I've created a fantastic database, named students_database with some amazing students ever had in the history.
Follow given steps to import it in your Oracle database:

  1. Click here to download the students_database.sql file.
  2. Open SQL*Plus command prompt and CONNECT to your database by logging in using SYSTEM username.
  3. Copy and paste the following command:
    • For Windows: 

      @C:\Users\[your_username]\Downloads\students_database.sql
      NOTE: Replace '[your_username]' by your user account name in Windows.

    • For Linux: 

      @/home/[your_username]/Downloads/students_database.sql
      NOTE: Replace '[your_username]' by your user account name in Linux.
  4. Press Enter key.
Now, you can query and practice SQL commands using students_database. There are two tables in this database named:
  1. stream(id, name)
  2. student(id, f_name, l_name, email, roll_no)
