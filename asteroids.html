shubham

This game was originally created Doug McInnes. His code can be found here, and you can play his version of the game online at his website. I modified the code and also replaced the iPad version of this game with android version.

If using a keyboard then Press Space to fire, the ← and → arrow keys to turn, and the ↑ arrow key to thrust.
Press P to pause, F to toggle the framerate display and hold down G to see a demonstration of the Grid System.
Hope, you'll enjoy it:)

Please rotate the device and reload the webpage to view the canvas!
